# GoalFocus – 专注目标桌面小工具（Windows）

GoalFocus 是一个运行在 **Windows 桌面** 的极简专注工具：  
你为自己设定一个 **长期目标**、一个 **当下目标**，再拆解出若干 **关键动作**，  
GoalFocus 会用一张悬浮在桌面的「专注卡片」陪你把这一件事做完。

## 核心功能

- 🎯 **目标分层**
  - 长期目标（例如：成为能自由使用英语工作的自己）
  - 当下目标（例如：今天完成一次 30 分钟口语练习）
  - 关键动作（可自由增删、拖拽排序）

- 🪧 **桌面悬浮卡片**
  - 无边框圆角卡片，始终置顶，可以自由拖拽、缩放
  - 鼠标移动到顶部区域时，显示标题栏和最小化 / 最大化 / 关闭按钮
  - 字体相对较大，目标和关键动作一目了然

- ✅ **关键动作管理**
  - 在「规划」页：
    - 输入框 + 回车快速添加关键动作
    - 双击空白新增行，双击文字可编辑
    - 拖拽调整优先级（自动编号 1/2/3…）
    - 右键删除单个动作
  - 在悬浮卡片中：
    - 勾选完成自动打勾 + 删除线
    - 双击空白新增动作，双击文字编辑
    - 拖拽调整顺序，右键删除动作
    - 一键【全选 / 全清】关键动作

- 🧨 **完成反馈与奖励**
  - 完成单个关键动作：
    - 居中出现小窗口动效（`success.gif`）
    - 播放音效（`sound.mp3`）
    - 显示奖杯图片（`pic.png`），用橙色文案条鼓励你继续
  - 完成整张专注卡片：
    - 全屏显示动效 GIF（透明覆盖，不遮蔽桌面）
    - 正中央显示奖杯图片 + 完成文案
    - 自动关闭悬浮卡片，回到「规划」页

- 📚 **归档 & 删除机会**
  - 每完成一张专注卡片，会记录到「归档」页：
    - 长期目标、当下目标
    - 创建时间、完成时间
    - 每个关键动作的完成时间
  - 每完成 5 张专注卡片，获得 1 次「删除机会」：
    - 可以在归档中选中一条卡片，消耗一次机会删除它
